# Privalia - Cotización de Seguro de Auto

Página de cotización de seguros desarrollada con HTML y Bootstrap 5.

## Estructura

```
├── index.html
├── css/
│   ├── global-styles.css
│   └── styles.css
├── svgs/
└── images/
```

## Tecnologías

- HTML5
- CSS3
- Bootstrap 5.3.2
- Open Sans (Google Fonts)

## Cómo usar

Abre el archivo `index.html` en tu navegador.

## Características

- Diseño responsive (mobile y desktop)
- Formulario de contacto con 4 campos
- Sección de comparación de compañías
- Sidebar con información del usuario (solo desktop)
- Usa principalmente clases de Bootstrap
